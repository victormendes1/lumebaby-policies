Política de Privacidade

LumeBaby - Assistente Inteligente para Cuidados com Bebê

Última atualização: Janeiro de 2025

Sua privacidade é nossa prioridade máxima. Esta política explica de forma clara e transparente como coletamos, usamos e protegemos seus dados.

1. Coleta de Dados

O LumeBaby coleta as seguintes categorias de informações para personalizar sua experiência:

  • Informações de Perfil: Nome do bebê, data de nascimento, foto (opcional)
  • Dados de Atividade: Padrões de sono, alimentação e marcos de desenvolvimento
  • Dados Biométricos: Frequência cardíaca do Apple Watch (opcional)
  • Dados de Autenticação: Apple ID para Sign in with Apple
  • Preferências: Configurações do aplicativo e notificações

2. Processamento Local

Privacidade por Design:

  • Todo áudio é processado localmente no seu dispositivo
  • Nenhum arquivo de áudio é enviado para servidores externos
  • Análise de choro usa apenas metadados (intensidade, duração)
  • Máxima privacidade para você e seu bebê

3. Uso de Dados

Utilizamos seus dados exclusivamente para:

  • Personalização: Recomendações adaptadas ao seu bebê
  • Melhoria da IA: Aprimorar a precisão das sugestões (com dados anônimos)
  • Analytics: Entender padrões de uso (dados agregados e anônimos)
  • Sincronização: Manter dados atualizados entre seus dispositivos via iCloud

4. Compartilhamento de Dados

Transparência total:

  • ❌ Não vendemos seus dados a terceiros
  • ❌ Não compartilhamos informações pessoais com anunciantes
  • ✅ Apenas metadados contextuais são enviados para a IA Claude (Anthropic)
  • ✅ Dados de família são compartilhados apenas com membros que você convidar
  • ✅ Conformidade total com LGPD (Lei Geral de Proteção de Dados)

5. Segurança

Implementamos as melhores práticas de segurança da Apple:

  • Criptografia end-to-end via CloudKit
  • Keychain para armazenamento seguro de credenciais
  • Sign in with Apple para autenticação segura
  • App Sandbox para isolamento de dados
  • Acesso restrito a dados sensíveis de saúde

6. Serviços de Terceiros

O LumeBaby utiliza os seguintes serviços:

  • Apple iCloud: Sincronização segura de dados entre dispositivos
  • Apple HealthKit: Integração com dados de saúde (com sua permissão)
  • Anthropic Claude API: Inteligência artificial para recomendações personalizadas

Todos os serviços são escolhidos por seus altos padrões de privacidade e segurança.

7. Seus Direitos (LGPD)

Você tem total controle sobre seus dados:

  • Acesso: Visualize todos os dados que coletamos sobre você
  • Correção: Atualize informações incorretas ou desatualizadas
  • Exclusão: Solicite a remoção completa de seus dados
  • Portabilidade: Exporte seus dados em formato legível
  • Revogação: Retire seu consentimento a qualquer momento

Para exercer seus direitos: Entre em contato através do email privacy@lumebaby.app

8. Retenção de Dados

Mantemos seus dados apenas pelo tempo necessário:

  • Dados ativos: enquanto você usar o aplicativo
  • Dados inativos: até 2 anos após última atividade
  • Backups: 90 dias para recuperação de desastres
  • Exclusão permanente: dentro de 30 dias após solicitação

9. Crianças e Menores

O LumeBaby é projetado para pais e cuidadores:

  • O app coleta dados sobre bebês, não de bebês
  • Usuários devem ter 18 anos ou mais
  • Cumprimos COPPA (Children's Online Privacy Protection Act)
  • Proteção especial para dados de crianças

10. Alterações nesta Política

Podemos atualizar esta política periodicamente:

  • Notificaremos sobre mudanças significativas via app
  • Data da última atualização sempre visível no topo
  • Versões anteriores disponíveis mediante solicitação
  • Uso continuado implica aceitação das mudanças

11. Contato

Dúvidas ou preocupações sobre sua privacidade?

  • Email: privacy@lumebaby.app
  • Encarregado de Dados (DPO): dpo@lumebaby.app
  • Website: lumebaby.app
